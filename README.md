# Android Study. https://vk.com/androidpnz

## Домашнее задание по занятию #6.

### Попроще

1. Создать первую активити (см. рисунок 1) с использованием любого менеджера разметки. Поле ввода должно принимать только номер телефона. При нажатии кнопки "Продолжить" запустить вторую активити (п. 2) явным интентом. Кнопки социальных сетей должны быть равномерно распределены по горизонтали. При нажатии на кнопку сети открыть для просмотра соответствующий URL ("https://vk.com", "https://facebook.com", "https://plus.google.com") неявным интентом.

2. Создать вторую активити (см. рисунок 2) с использованием любого менеджера разметки. Поле ввода должно принимать только цифры. При нажатии кнопки "Войти" запустить третью активити. 

3. Создать третью активити (см. рисунок 3) с полем ввода и списком контактов. Для вывода списка контактов использовать ListView и любой подходящий адаптер. Создать класс Person с тремя полями: имя (String), телефон (String), фото (int). В активити создайте массив с 20-30 объектами класса Person, имена и телефоны рандомные, в качестве фото укажите идентификатор любой иконки из ресурсов приложения (например, ic_launcher, или положите в ресурсы любую другую). Выведите этот массив в ListView.

### Посложнее

1. В разметке активити первого шага авторизации использовать ConstraintLayout.

2. В разметке второй активити использовать CoordinatorLayout, при нажатии кнопки "Войти" выводить snackbar с текстом "Я молодец!" и открывать третью активити.

3. В третьей активити использовать RecyclerView вместо ListView, реализовать ViewHolder.

Работа с репозиторием аналогично предыдущим заданиям: новая ветка в локальном репозитории, коммит в неё, затем пуш на гитхаб...
